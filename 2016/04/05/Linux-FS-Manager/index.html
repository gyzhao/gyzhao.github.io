Linux 基本的文件管理操作

Linux
  • 文件管理

这里介绍了在Linux中如何使用命令来对文件进行最基础的文件管理操作。

复制文件和目录

在Linux中使用命令cp来复制文件或者目录，使用方式：

cp 源文件(文件夹)  目标文件(文件夹)

cp命令常用参数：

  • -r 递归复制整个目录
  • -v 显示详细信息

移动、重命名一个文件或目录

使用mv命令来移动或者重命名一个文件夹以及目录，使用方式：

mv 源文件(文件夹) 目标目录

移动时如果指定文件名，则会进行重命名

删除文件或目录

使用命令rm用以删除一个文件或文件夹，使用方式：

rm 文件/文件夹

rm命令常用参数：

  • -i 交互式删除
  • -r 递归删除文件夹下所有文件
  • -f 强制删除，不提示（慎用）

创建和删除空目录

使用命令mkdir可以创建一个空目录

mkdir 目录名称

使用命令rmdir可以删除一个空目录

rmdir 目录名称

如果是需要删除包含文件的目录的话还是要使用rm -r命令
